PyCon India 2020 Handbook
=========================

Introduction
------------

This purpose of this handbook is to familiarize everyone with the details, process
decided by the team of the conference. It is recommended that all the volunteers,
as well as the organizers read through this document and update when required.

The aim with this document is to make the process as smooth as possible for everyone
and the attendees, speakers, and the sponsors having a good time during the conference.

Working Groups
--------------

PyCon India 2020 is divided into various Working Groups (WG)

- Call For Proposals WG
- Code of Conduct WG
- Content WG
- Design WG
- Dev Sprints WG
- Diversity WG
- Mentorship WG
- Platform WG
- Review Panel WG
- Social Media WG
- Sponsorship WG
- Technology WG
- Tickets WG

Events
------

PyCon India 2020 will have the following set of events during the span of four days

- Keynotes
- Talks
- Diversity Events
- Birds of Feathers (BoF)
- .extend()
- Workshops
- Dev Sprints

Stack Knowledge Base
--------------------

PyCon India 2020 is an online event, built on a stack of multiple platforms, vendor partners.

- Github: Hosts website, blog, assets, and badges
- TownScript: Ticketing Vendor
- Hopin: The event platform that would host PyCon India 2020
- Twitter, Facebook, Linkedin: Social Media Platform.
